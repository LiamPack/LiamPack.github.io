The graduate core course in analysis at Cornell. 

Course textbook: [Real Analysis: Measure Theory, Integration and Hilbert Spaces](https://www.amazon.com/Real-Analysis-Integration-Princeton-Lectures-ebook/dp/B007BOK6PW).

Recitation covered applications or extensions of material from class. Some topics:
+ Hahn-Banach, Convex Duality ([Brezis](https://link.springer.com/book/10.1007/978-0-387-70914-7) \S 1)
+ Almost Every Number is Normal via probability ([Kac; Statistical Independence in Probability, Analysis, and Number Theory](https://bookstore.ams.org/car-12/) \S 1-2)
+ Jordan measure decomposition & Radon-Nikodym
+ Some applications of Jensen's inequality
+ $(L^p)^* = L^q$, and how a weakly-convergent sequence can fail to
  converge strongly
+ Basic $L^2$ theory for $\Delta u = f$ ([Krylov; Lectures on Elliptic and Parabolic Equations in Sobolev Spaces](https://bookstore.ams.org/view?ProductCode=GSM%2F96) \S 1)
